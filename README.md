# Chicken Shop Digital Bill Book (Full Stack)

This is a MERN-based, mobile-first billing web app built from your PRD and architecture docs.

## Stack
- Frontend: React + Vite
- Backend: Node.js + Express
- Database: MongoDB + Mongoose
- Auth: JWT + bcrypt

## MVP Modules
- Owner login
- Dashboard (today bills, revenue, customers, pending)
- Customers (add, list, search, detail)
- Bills (create, list)
- Pending balances (sorted)

## Default Login
- Username: `owner`
- Password: `owner123`

## Run
1. Copy `.env.example` to `.env` at repo root and update values.
2. Install dependencies from repo root.
3. Start dev servers from repo root.

The frontend runs on port 5173, backend on port 5000 by default.
